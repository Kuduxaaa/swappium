# Swappium

## Installation
```bash
git clone https://github.com/Kuduxaaa/swappium
cd swappium
npm install
npm audit fix
composer install

# Change database information in .env file

php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan passport:install
php artisan serve

# Open new termianl tab

npm run dev
```
